# Medbotic Frontend

Este repositorio contiene el Frontend de Medbotic, una aplicación web que permite a los usuarios interactuar con un ChatBot diseñado para asistencia médica automatizada. La interfaz está construida para ser sencilla, rápida y flexible, ofreciendo una experiencia limpia para consultas y flujos conversacionales.

## 🚀 Características Clave
- Interfaz de Chat optimizada para interacción fluida con el modelo backend.
- Manejo de sesiones para mantener el contexto de conversación.
- Diseño responsivo, funciona en desktop y móvil.
- Integración directa con la API de Medbotic, usando variables de entorno para mayor seguridad.
- Componentes reutilizables y estructura organizada para facilitar mantenimiento y escalabilidad.
- Soporte para diferentes modelos o endpoints, según la configuración del entorno.
---

## ⚙️ Configuración de Entornos
El proyecto utiliza variables de entorno para garantizar seguridad y flexibilidad.

- Usa el archivo `.env.example` como plantilla.
- Copia y renombra:

```bash
cp .env.example .env
```

## 📦 Instalación y Uso

#### 1. Instalar dependencias
```bash
npm install
```
#### 2. Iniciar servidor de desarrollo

```bash
npm run dev
```

### 3. Construir para producción

```bash
npm run dev
```
npm run build

4. Previsualizar build
```bash
npm run preview
```
